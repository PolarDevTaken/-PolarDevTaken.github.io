10. < br > Tworzy jedną linijkę przerwy
11. < cite > Powinien być wykorzystywany aby zdefiniować źródło informacji
12. < abbr > Reprezentuje akronimy
13. < samp > Oznaczamy przkład kodu komputerowego

16. < input > Tworzymy pola w które użytkownik może umieszcić dane
17. < bdo > Nadpisujemy kierunek tekstu
18. < label > Etykieta

Elementy Blokowe
NAZWA ELEMENTU OPIS
1. < h1-h6 > Nagłówki na stronie
2. < div > Kontener, odziela sekcje strony
3. < p > Paragraf
4. < nav > Kontener w którym wstawiamy linki do nawigowania po stronie
5. < main > Kontener w którym wstawiamy główną treść strony
6. < section > Sekcja
7. < article > Umieszczamy w nim artykuł
8. < aside > Wstawiamy w nim treść niezwiązaną ze stroną
9. < footer > Znacznik używany na dole strony, często dane kontaktowe są tutaj umieszczane
10. < header > Kontener który jest "wstępem" do szkoły. Tutaj umieszczamy np. baner strony
11. < blockquote > Pojemnik na cytat
12. < table > Tworzenie tabeli
13. < video > Umieszczamy w nim pliki wideo.
14. < ol > Lista która jest uporządkowana
15. < ul > Lista która nie jest uporządkowana
16. < dl > Lista opisu
17. < dd > Opis elementu dla dl
18. < dt > Nazwa elementu z dl
19. < li > Jeden punkt z listy ul/ol
20. < address > Tutaj umieszczamy dane kontaktowe dla osoby lub organizacji dokumentu/artykułu.
