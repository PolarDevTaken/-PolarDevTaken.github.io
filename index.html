Elementy Liniowe
lp. nazwa opis
1. sup gorny indek
2. sub dolny indeks
3. b bogrubienie
4. i kursywa
5. strong ag służy do definiowania tekstu o dużym znaczeniu, zawartość wewnątrz jest zazwyczaj wyświetlana pogrubioną czcionką.
6. span to wbudowany kontener służący do oznaczania części tekstu lub części dokumentu.
7. q definiuje krótki cytat.
8. img tag służy do osadzania obrazu na stronie HTML.
9. a definiuje hiperłącze, które służy do łączenia z jednej strony na drugą.
10. br wstawia pojedynczy podział wiersza.
11. cite tag określa tytuł dzieła twórczego (np. książki, wiersza, piosenki, filmu, obrazu, rzeźby itp.).
12. abbr tag definiuje skrót lub akronim, taki jak "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM"
13. samp służy do definiowania przykładowych danych wyjściowych z programu komputerowego
14. code służy do definiowania fragmentu kodu komputerowego, zawartość wewnątrz jest wyświetlana w domyślnej czcionce monospace przeglądarki.
15. select służy do tworzenia listy rozwijanej.
16. input określa pole wejściowe, w którym użytkownik może wprowadzać dane.
17. bdo tag służy do zastąpienia bieżącego wyrównywania tekstu.
18. label tag definiuje etykietę dla kilku elementów
19. time definiuje określoną godzinę (lub datę/ godzinę).
20. small znacznik zmniejszajacy tekst w srodku
